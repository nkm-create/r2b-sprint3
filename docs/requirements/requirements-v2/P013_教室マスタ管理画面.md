## 4. 機能詳細

### F120: 教室一覧表示

#### 出力項目

| 項目名 | DBカラム/計算 | 説明 |
|--------|-------------|------|
| 教室名 | classrooms.classroom_name | 教室の名称 |
| 教室コード | classrooms.classroom_code | 一意の識別コード |
| 所属エリア | classrooms.area_id | エリア名 |
| 住所 | classrooms.address | 教室の住所 |
| 電話番号 | classrooms.phone_number | 連絡先 |
| ステータス | classrooms.status | operating（運営中）/closed（閉鎖） |
| 教室長 | users.user_name | 担当教室長 |
| 講師数 | COUNT(teachers) | 登録講師数 |
| 生徒数 | COUNT(students) | 登録生徒数 |

#### API仕様

```
GET /api/classrooms

クエリパラメータ:
- filter[area_id]: エリアIDで絞り込み
- filter[status]: operating / closed / all
- search: 教室名・教室コードの部分一致
- page, per_page: ページネーション
- sort: classroom_name / classroom_code / -created_at

レスポンス:
{
  "data": [
    {
      "classroom_id": "uuid",
      "classroom_name": "渋谷教室",
      "classroom_code": "SBY001",
      "area_id": "tokyo",
      "area_name": "東京エリア",
      "address": "東京都渋谷区...",
      "phone_number": "03-1234-5678",
      "status": "operating",
      "manager": {
        "user_id": "uuid",
        "user_name": "山田 太郎"
      },
      "teacher_count": 25,
      "student_count": 80
    }
  ],
  "pagination": {...}
}
```

#### SQL例

```sql
SELECT
  c.classroom_id,
  c.classroom_name,
  c.classroom_code,
  c.area_id,
  c.address,
  c.phone_number,
  c.status,
  u.user_id as manager_id,
  u.user_name as manager_name,
  COALESCE(t_count.count, 0) as teacher_count,
  COALESCE(s_count.count, 0) as student_count
FROM classrooms c
LEFT JOIN (
  SELECT uc.classroom_id, u.user_id, u.user_name
  FROM user_classrooms uc
  JOIN users u ON uc.user_id = u.user_id
  WHERE u.role = 'classroom_manager'
    AND u.is_active = true
    AND u.deleted_at IS NULL
    AND uc.deleted_at IS NULL
) u ON c.classroom_id = u.classroom_id
LEFT JOIN (
  SELECT classroom_id, COUNT(*) as count
  FROM teachers
  WHERE is_current = true AND deleted_at IS NULL
  GROUP BY classroom_id
) t_count ON c.classroom_id = t_count.classroom_id
LEFT JOIN (
  SELECT classroom_id, COUNT(*) as count
  FROM students
  WHERE is_current = true AND deleted_at IS NULL
  GROUP BY classroom_id
) s_count ON c.classroom_id = s_count.classroom_id
WHERE c.deleted_at IS NULL
  AND (:area_id IS NULL OR c.area_id = :area_id)
  AND (:status IS NULL OR c.status = :status)
  AND (:search IS NULL OR c.classroom_name ILIKE '%' || :search || '%' OR c.classroom_code ILIKE '%' || :search || '%')
ORDER BY c.classroom_name
LIMIT :per_page OFFSET :offset;
```

### F121: 教室新規作成

#### 入力項目

| 項目名 | 型 | 必須 | DBカラム | バリデーション |
|--------|-----|------|---------|--------------|
| 教室名 | string | ○ | classroom_name | 最大100文字 |
| 教室コード | string | ○ | classroom_code | 英数字、最大20文字、重複不可 |
| 所属エリア | string | ○ | area_id | エリアマスタから選択 |
| 住所 | string | - | address | 最大200文字 |
| 電話番号 | string | - | phone_number | 電話番号形式 |
| ステータス | enum | ○ | status | classroom_status（operating/closed）、デフォルト: operating |

#### API仕様

```
POST /api/classrooms

リクエスト:
{
  "classroom_name": "新宿教室",
  "classroom_code": "SJK001",
  "area_id": "tokyo",
  "address": "東京都新宿区...",
  "phone_number": "03-9876-5432",
  "status": "operating"
}

成功レスポンス (201 Created):
{
  "classroom_id": "uuid",
  "message": "教室を作成しました"
}
```

#### トランザクション処理

```sql
BEGIN;

-- 教室作成
INSERT INTO classrooms (
  classroom_id, classroom_name, classroom_code, area_id,
  address, phone_number, status
)
VALUES (
  gen_random_uuid(), :classroom_name, :classroom_code, :area_id,
  :address, :phone_number, :status
)
RETURNING classroom_id;

-- 教室設定をデフォルト値で作成（classroom_idがPK）

  classroom_id, booth_count, weekday_slots, saturday_slots, operating_days

  (gen_random_uuid(), :classroom_id, 'weekday', 3, '19:00', '20:20'),

  (gen_random_uuid(), :classroom_id, 'saturday', 1, '13:00', '14:20'),
```

### F122: 教室情報編集

#### API仕様

```
PUT /api/classrooms/{classroom_id}

リクエスト:
{
  "classroom_name": "新宿教室（移転後）",
  "area_id": "tokyo",
  "address": "東京都新宿区（新住所）...",
  "phone_number": "03-1111-2222",
  "status": "operating"
}

成功レスポンス (200 OK):
{
  "classroom_id": "uuid",
  "message": "教室情報を更新しました"
}

警告付きレスポンス（閉鎖時、200 OK）:
{
  "classroom_id": "uuid",
  "message": "教室情報を更新しました",
  "warning": "この教室には25名の講師と80名の生徒が登録されています"
}
```

#### 制約

| 制約 | 条件 | 動作 |
|------|------|------|
| 教室コード変更不可 | 常時 | リクエストから除外、または変更時エラー |
| 閉鎖時の警告 | status = 'closed' かつ 講師/生徒あり | 警告を表示、確認後に変更可能 |

### F123: 教室削除

#### API仕様

```
DELETE /api/classrooms/{classroom_id}

成功レスポンス (200 OK):
{
  "message": "教室を削除しました"
}

警告付き削除確認（データあり時、200 OK with confirmation required）:
{
  "require_confirmation": true,
  "warning": {
    "message": "この教室には以下のデータが存在します",
    "details": {
      "teacher_count": 25,
      "student_count": 80,
      "schedule_count": 5,
      "confirmed_schedules": 3
    },
    "suggestion": "データの移行または閉鎖をお勧めします"
  }
}

確認後の削除:
DELETE /api/classrooms/{classroom_id}?confirm=true
```

#### 処理分岐

| 条件 | 動作 |
|------|------|
| データなし | 即時論理削除 |
| 講師・生徒データあり | 警告表示、確認後に削除可能 |
| 確定済み時間割あり | 警告表示（データ移行推奨）、確認後に削除可能 |

#### トランザクション処理

```sql
BEGIN;

-- 関連データの確認
SELECT
  (SELECT COUNT(*) FROM teachers WHERE classroom_id = :classroom_id AND deleted_at IS NULL) as teacher_count,
  (SELECT COUNT(*) FROM students WHERE classroom_id = :classroom_id AND deleted_at IS NULL) as student_count,
  (SELECT COUNT(*) FROM schedules WHERE classroom_id = :classroom_id) as schedule_count,
  (SELECT COUNT(*) FROM schedules WHERE classroom_id = :classroom_id AND status = 'confirmed') as confirmed_schedules;

-- 確認済みの場合、論理削除
UPDATE classrooms SET
  deleted_at = NOW(),
  classroom_code = classroom_code || '_deleted_' || NOW()::text  -- コード重複防止
WHERE classroom_id = :classroom_id;

-- 関連するuser_classroomsも論理削除
UPDATE user_classrooms SET
  deleted_at = NOW()
WHERE classroom_id = :classroom_id;

-- 監査ログ
INSERT INTO audit_logs (user_id, action, entity_type, entity_id, old_value)
VALUES (:user_id, 'DELETE', 'Classroom', :classroom_id, :old_value_json);

COMMIT;
```
