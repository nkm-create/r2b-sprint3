## 4. 機能詳細

### F030: 時間枠設定

#### 入力項目

| 項目名 | 型 | DBカラム | 説明 |
|--------|-----|---------|------|
| 曜日種別 | enum | day_type | weekday / saturday |
| 時限番号 | number | slot_number | 1, 2, 3... |
| 開始時刻 | time | start_time | HH:MM形式 |
| 終了時刻 | time | end_time | HH:MM形式 |

#### API仕様

```
GET /api/classrooms/{classroom_id}/time-slots

レスポンス:
{
  "weekday_slots": [
    {"slot_number": 1, "start_time": "16:00", "end_time": "17:20"},
    {"slot_number": 2, "start_time": "17:30", "end_time": "18:50"},
    ...
  ],
  "saturday_slots": [...]
}

PUT /api/classrooms/{classroom_id}/time-slots

リクエスト:
{
  "weekday_slots": [...],
  "saturday_slots": [...]
}
```

#### トランザクション処理

```sql
BEGIN;

-- 既存の時間枠を削除
DELETE FROM time_slots WHERE classroom_id = :classroom_id;

-- 新しい時間枠を登録
INSERT INTO time_slots (time_slot_id, classroom_id, day_type, slot_number, start_time, end_time)
VALUES (gen_random_uuid(), :classroom_id, :day_type, :slot_number, :start_time, :end_time);

-- classroom_settingsを更新
UPDATE classroom_settings SET
  weekday_slots = :weekday_count,
  saturday_slots = :saturday_count,
  updated_at = NOW()
WHERE classroom_id = :classroom_id;

-- 監査ログ
INSERT INTO audit_logs (user_id, action, entity_type, entity_id, old_value, new_value)
VALUES (:user_id, 'UPDATE', 'TimeSlots', :classroom_id, :old_value, :new_value);

COMMIT;
```

### F034: Google Form連携設定

#### API仕様

**連携設定の追加**:
```
POST /api/classrooms/{classroom_id}/google-form-connections

リクエスト:
{
  "data_type": "teacher_shift",
  "spreadsheet_url": "https://docs.google.com/spreadsheets/d/xxx",
  "sheet_name": "シフト希望",
  "column_mapping": {
    "name": "A",
    "day_of_week": "B",
    "slot_number": "C",
    "preference_value": "D"
  }
}
```

**OAuth認証フロー**:
```
GET /api/auth/google/authorize
→ Google OAuth 2.0 認証画面へリダイレクト

GET /api/auth/google/callback?code=xxx
→ アクセストークン取得、暗号化して保存
```

#### カラムマッピング詳細

**講師シフト希望**:

| システム項目 | 説明 | column_mappingキー |
|-------------|------|------------------|
| 氏名 | 講師の氏名 | name |
| 曜日 | 希望曜日 | day_of_week |
| 時限 | 希望時限 | slot_number |
| 希望値 | ○/× | preference_value |

**生徒受講希望**:

| システム項目 | 説明 | column_mappingキー |
|-------------|------|------------------|
| 氏名 | 生徒の氏名 | name |
| 曜日 | 希望曜日 | day_of_week |
| 時限 | 希望時限 | slot_number |
| 希望値 | ○/×/△ | preference_value |

#### エラーハンドリング

| エラー種別 | エラーコード | メッセージ |
|-----------|------------|----------|
| 認証エラー | GF_001 | 再認証が必要です |
| スプレッドシート未検出 | GF_002 | スプレッドシートが見つかりません |
| シート未検出 | GF_003 | シートが見つかりません |
| 権限エラー | GF_004 | アクセス権限がありません |
| レート制限 | GF_005 | しばらく待ってから再試行してください |
| マッピング不整合 | GF_006 | カラムマッピングが不正です |

## 7. エラーコード

| コード | メッセージ | 条件 |
|--------|----------|------|
| VAL_030 | 時間枠の開始時刻は終了時刻より前に設定してください | 時刻不整合 |
| VAL_031 | 時間枠が重複しています | 時間帯重複 |
| VAL_032 | ブース数は1以上で入力してください | 範囲外 |
| VAL_033 | 営業曜日を1日以上選択してください | 未選択 |
