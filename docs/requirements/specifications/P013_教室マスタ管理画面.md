# 仕様書: 教室マスタ管理画面

## 基本情報


| 項目     | 内容        |
| ------ | --------- |
| ページID  | P013      |
| ページ名   | 教室マスタ管理画面 |
| 対象ユーザー | システム管理者   |


## 目的

システムで管理する教室のマスタ情報を登録・編集・管理する。

## 機能一覧


| 機能ID | 機能       | 説明             |
| ------ | -------- | -------------- |
| F120   | 教室一覧表示   | 登録済み教室の一覧を表示   |
| F121   | 教室新規作成   | 新しい教室を登録       |
| F122   | 教室情報編集   | 既存教室の情報を編集     |
| F123   | 教室削除     | 教室を削除（論理削除）    |


## UI要素


| 要素       | 説明              | 位置        |
| -------- | --------------- | --------- |
| ヘッダー     | 共通ヘッダー（管理者用）    | 上部固定      |
| サイドバー    | 管理者用サイドバー       | 左側固定      |
| 検索バー     | 教室名で検索          | メインエリア上部左 |
| フィルターボタン | エリア・ステータスでフィルター | 検索バー横     |
| 新規作成ボタン  | 教室作成モーダルを開く     | メインエリア上部右 |
| 教室一覧テーブル | 教室情報の一覧         | メインエリア中部  |
| エリアバッジ   | 所属エリアを表示        | テーブル内     |
| 編集ボタン    | 各行の編集ボタン        | テーブル内     |
| 削除ボタン    | 各行の削除ボタン        | テーブル内     |
| ページネーション | ページ切り替え         | テーブル下部    |


## 機能とUIの結びつき


| 機能       | UI要素          | インタラクション           |
| -------- | ------------- | ------------------ |
| 教室一覧表示   | 教室一覧テーブル      | 一覧表示、ソート可能         |
| 教室新規作成   | 新規作成ボタン       | クリックでモーダル表示、入力後保存  |
| 教室情報編集   | 編集ボタン         | クリックでモーダル表示、編集後保存  |
| 教室削除     | 削除ボタン         | クリックで確認ダイアログ、確定で削除 |
| エリア設定    | 作成・編集モーダル内    | エリアを選択             |
| 検索・フィルター | 検索バー、フィルターボタン | 入力・選択で一覧を絞り込み      |


## 教室作成・編集モーダルの項目


| 項目    | 必須  | 説明               |
| ----- | --- | ---------------- |
| 教室名   | ○   | 教室の名称            |
| 教室コード | ○   | システム内で使用する一意のコード |
| 所属エリア | ○   | エリアを選択           |
| 住所    | -   | 教室の住所            |
| 電話番号  | -   | 教室の電話番号          |
| ステータス | ○   | 運営中 / 閉鎖         |


## 遷移


| 遷移元           | 遷移先           | トリガー              |
| ------------- | ------------- | ----------------- |
| P012 ユーザー管理画面 | -             | サイドバー「教室管理」クリック   |
| -             | P012 ユーザー管理画面 | サイドバー「ユーザー管理」クリック |


## 備考

- 教室削除は関連データ（講師、生徒、時間割など）がある場合は警告表示
- 削除は論理削除、復元可能
- エリアはプルダウンで選択（エリアマスタは将来対応）
