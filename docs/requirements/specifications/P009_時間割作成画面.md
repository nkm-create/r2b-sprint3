# 仕様書: 時間割作成画面

## 基本情報

| 項目 | 内容 |
|-----|------|
| ページID | P009 |
| ページ名 | 時間割作成画面 |
| 対象ユーザー | 教室長 |

## 目的

AI最適化による時間割生成を実行し、結果を確認・調整・確定する。生成から調整、PDF出力までを一画面で完結させる。

## 備考

- 生成処理はバックグラウンドで実行、途中キャンセル可能
- ドラッグ&ドロップ時は制約違反をリアルタイムでチェック
- 注意点は「1対1になっているコマ」「講師の連続授業超過」などを表示
- PDF出力は全体/講師別（全員or選択）/生徒別（全員or選択）を選択可能

## アルゴリズム選択機能

### 選択可能なアルゴリズム

| ID | アルゴリズム | 適用ケース | 特徴 |
|----|------------|-----------|------|
| ALG001 | 遺伝的アルゴリズム (GA) | 中〜大規模、柔軟な制約 | 大域的探索、多様な解 |
| ALG002 | 制約充足ソルバー (CSP) | ハード制約が多い | 制約の完全充足を保証 |
| ALG003 | 整数線形計画法 (ILP) | 小〜中規模 | 数学的に最適解を保証 |
| ALG004 | 焼きなまし法 (SA) | 連続的な改善 | 局所最適解からの脱出 |
| ALG005 | 貪欲法+局所探索 (Greedy+LS) | 大規模、時間制約が厳しい | 高速、実用的な解 |
| ALG006 | ハイブリッド (Hybrid) | 複雑な問題 | 複数アルゴリズムの長所を組合せ |

### 選択モード

| モード | 説明 |
|--------|------|
| 自動選択 | 問題規模・制約密度・時間予算・品質要求に基づき最適なアルゴリズムをシステムが推奨・選択 |
| 手動選択 | ユーザーがドロップダウンから任意のアルゴリズムを選択 |

### アルゴリズム自動選択の判断基準

- **問題規模**: 講師数・生徒数・コマ数から算出
- **制約密度**: ハード制約・ソフト制約の数と複雑さ
- **時間予算**: ユーザー指定の生成時間上限
- **品質要求**: 充足率目標値

## AIエージェント機能詳細

### F087: AIエージェント対話

チャット形式でエージェントと対話しながら時間割を調整する。

**対話例**:
```
ユーザー: 「月曜3限の佐藤先生のコマを火曜に移動できる？」
エージェント: 「火曜1限と2限が空いています。1限は生徒Aも空いています（推奨）」
ユーザー: 「1限でお願い」
エージェント: 「移動しました。充足率: 88.5% → 89.2%」
```

**チャットパネル機能**:
- 自然言語での調整指示入力
- エージェントからの提案・確認メッセージ表示
- 操作履歴のスクロール表示
- 提案に対する「採用」「却下」ボタン

### F088: AIエージェント思考過程表示

エージェントの判断過程をステップ表示する。

**表示ステップ**:
| ステップ | 表示内容 |
|---------|---------|
| 初期化 | データロード状況 |
| 問題分析 | 問題特性の分析結果（規模、制約密度等） |
| アルゴリズム選択 | 選択されたアルゴリズムと理由 |
| 最適化実行 | 現在の世代数、充足率、制約違反数 |
| 評価 | 解の評価結果と改善余地 |
| 改善ループ | 改善施策と結果（最大3回） |
| 確定準備 | 最終結果サマリー |

**進捗表示**:
- 各ステップのインジケーター（待機中/実行中/完了）
- 現在ステップのハイライト表示
- 予測残り時間の表示

### F089: AIエージェント手動介入

生成処理中にユーザーが介入できる機能。

**介入オプション**:
| オプション | 説明 |
|-----------|------|
| 一時停止 | 現在の処理を一時停止し、状況を確認 |
| パラメータ調整 | 充足率目標、時間予算、重み付け等を変更して再開 |
| 手動編集 | 現在の途中結果に対してドラッグ&ドロップで手動調整 |
| やり直し | 条件設定画面に戻って再生成 |
| 続行 | 一時停止を解除して処理を続行 |

**パラメータ調整パネル**:
- 充足率目標値スライダー
- 時間予算入力
- 制約優先度の重み付け調整
- アルゴリズム変更オプション
