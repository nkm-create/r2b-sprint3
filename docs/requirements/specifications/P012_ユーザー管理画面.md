# 仕様書: ユーザー管理画面

## 基本情報

| 項目 | 内容 |
|-----|------|
| ページID | P012 |
| ページ名 | ユーザー管理画面 |
| 対象ユーザー | システム管理者 |

## 目的

システムを利用するユーザー（教室長、エリアマネジャー、システム管理者）のアカウントを管理する。

## 機能一覧

| 機能ID | 機能 | 説明 |
|--------|------|------|
| F110 | ユーザー一覧表示 | 登録済みユーザーの一覧を表示 |
| F111 | ユーザー新規作成 | 新しいユーザーアカウントを作成 |
| F112 | ユーザー情報編集 | 既存ユーザーの情報を編集 |
| F113 | ユーザー削除 | ユーザーを削除（論理削除） |
| F114 | パスワードリセット（管理者） | ユーザーのパスワードをリセット |

## UI要素

| 要素 | 説明 | 位置 |
|------|------|------|
| ヘッダー | 共通ヘッダー（管理者用） | 上部固定 |
| サイドバー | 管理者用サイドバー | 左側固定 |
| 検索バー | ユーザー名・メールで検索 | メインエリア上部左 |
| フィルターボタン | 役割・ステータスでフィルター | 検索バー横 |
| 新規作成ボタン | ユーザー作成モーダルを開く | メインエリア上部右 |
| ユーザー一覧テーブル | ユーザー情報の一覧 | メインエリア中部 |
| 役割バッジ | 教室長/エリアマネジャー/管理者を色分け表示 | テーブル内 |
| 編集ボタン | 各行の編集ボタン | テーブル内 |
| 削除ボタン | 各行の削除ボタン | テーブル内 |
| リセットボタン | パスワードリセットボタン | テーブル内 |
| ページネーション | ページ切り替え | テーブル下部 |

## 機能とUIの結びつき

| 機能 | UI要素 | インタラクション |
|------|--------|----------------|
| ユーザー一覧表示 | ユーザー一覧テーブル | 一覧表示、ソート可能 |
| ユーザー新規作成 | 新規作成ボタン | クリックでモーダル表示、入力後保存 |
| ユーザー情報編集 | 編集ボタン | クリックでモーダル表示、編集後保存 |
| ユーザー削除 | 削除ボタン | クリックで確認ダイアログ、確定で削除 |
| 権限設定 | 作成・編集モーダル内 | 役割選択で権限を設定 |
| 担当教室設定 | 作成・編集モーダル内 | 教室を選択（複数可） |
| パスワードリセット | リセットボタン | クリックで確認後、リセットメール送信 |
| 検索・フィルター | 検索バー、フィルターボタン | 入力・選択で一覧を絞り込み |

## ユーザー作成・編集モーダルの項目

| 項目 | 必須 | 説明 |
|------|------|------|
| 氏名 | ○ | ユーザーの氏名 |
| メールアドレス | ○ | ログイン用メールアドレス |
| 役割 | ○ | 教室長 / エリアマネジャー / システム管理者 |
| 担当教室 | △ | 教室長・エリアマネジャーの場合必須 |
| ステータス | ○ | 有効 / 無効 |

## 遷移

| 遷移元 | 遷移先 | トリガー |
|--------|--------|----------|
| P001 ログイン画面 | - | ログイン成功（システム管理者の場合） |
| - | P013 教室マスタ管理画面 | サイドバー「教室管理」クリック |

## 備考

- 新規作成時は初期パスワードをメールで送信
- 削除は論理削除、復元可能
- 自分自身は削除不可
