# 仕様書: ターム管理画面

## 基本情報


| 項目     | 内容      |
| ------ | ------- |
| ページID  | P011    |
| ページ名   | ターム管理画面 |
| 対象ユーザー | 教室長     |


## 目的

学期・タームごとの時間割を一覧管理し、過去タームの参照・複製・編集を行う。

## 機能一覧


| 機能ID | 機能      | 説明                |
| ------ | ------- | ----------------- |
| F100   | ターム一覧表示 | 登録済みタームを一覧表示      |
| F101   | ターム詳細確認 | 特定タームの時間割サマリーを確認  |
| F102   | ターム複製   | 既存タームをベースに新タームを作成 |
| F103   | ターム編集   | タームの時間割調整画面へ遷移    |
| F104   | ターム削除   | 不要なタームを削除         |


## UI要素


| 要素        | 説明                        | 位置        |
| --------- | ------------------------- | --------- |
| ヘッダー      | 共通ヘッダー                    | 上部固定      |
| サイドバー     | 共通サイドバー                   | 左側固定      |
| 検索バー      | ターム名で検索                   | メインエリア上部左 |
| 期間フィルター   | 期間で絞り込み                   | 検索バー横     |
| 新規作成ボタン   | 条件設定画面へ遷移（新規ターム）          | メインエリア上部右 |
| ターム一覧テーブル | ターム情報の一覧（名前、期間、充足率、ステータス） | メインエリア中部  |
| ステータスバッジ  | 作成中/確定/アーカイブなどのステータス      | テーブル内     |
| 詳細ボタン     | ターム詳細パネルを開く               | テーブル内     |
| 複製ボタン     | タームを複製して条件設定画面へ           | テーブル内     |
| 編集ボタン     | 時間割作成画面へ遷移                | テーブル内     |
| 削除ボタン     | ターム削除（確認ダイアログあり）          | テーブル内     |
| ターム詳細パネル  | 選択タームのサマリー情報              | 右側スライドイン  |


## 機能とUIの結びつき


| 機能      | UI要素           | インタラクション                 |
| ------- | -------------- | ------------------------ |
| ターム一覧表示 | ターム一覧テーブル      | 一覧表示、ソート可能               |
| ターム検索   | 検索バー、期間フィルター   | 入力・選択で絞り込み               |
| ターム詳細確認 | 詳細ボタン、ターム詳細パネル | クリックで右パネルに詳細表示           |
| ターム複製   | 複製ボタン          | クリックで条件設定画面へ遷移（前回条件をコピー） |
| ターム編集   | 編集ボタン          | クリックで時間割作成画面へ遷移          |
| ターム削除   | 削除ボタン          | クリックで確認ダイアログ、確定で削除       |


## ターム詳細パネルの表示項目


| 項目    | 説明                 |
| ----- | ------------------ |
| ターム名  | タームの名称             |
| 期間    | 開始日〜終了日            |
| ステータス | 作成中 / 確定 / アーカイブ   |
| 充足率   | 1対2の充足率            |
| コマ数   | 総コマ数、1対2コマ数、1対1コマ数 |
| 作成日   | ターム作成日時            |
| 最終更新日 | 最後に編集した日時          |


## 遷移


| 遷移元          | 遷移先          | トリガー              |
| ------------ | ------------ | ----------------- |
| P002 ダッシュボード | -            | サイドバー「ターム管理」クリック  |
| -            | P008 条件設定画面  | 「新規作成」ボタン、「複製」ボタン |
| -            | P009 時間割作成画面 | 「編集」ボタン           |


## 備考

- 確定済みタームは編集に警告を表示（変更は振替対応画面で行うことを推奨）
- アーカイブ済みタームは閲覧のみ
- 削除は論理削除、復元可能
