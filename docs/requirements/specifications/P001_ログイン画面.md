# 仕様書: ログイン画面

## 基本情報

| 項目 | 内容 |
|-----|------|
| ページID | P001 |
| ページ名 | ログイン画面 |
| 対象ユーザー | 教室長、エリアマネジャー、システム管理者 |

## 目的

システムへの認証を行い、ユーザーの役割に応じた画面へ遷移させる。

## 機能一覧

| 機能ID | 機能 | 説明 |
|--------|------|------|
| F001 | ログイン | メールアドレスとパスワードでシステムにログインする |
| F002 | パスワードリセット | パスワードを忘れた場合にリセット用メールを送信する |
| F003 | ログイン状態維持 | 次回アクセス時に自動ログインする設定 |

## UI要素

| 要素 | 説明 | 位置 |
|------|------|------|
| ロゴ | システムロゴ | 上部中央 |
| メールアドレス入力 | ログイン用メールアドレスを入力 | フォーム上部 |
| パスワード入力 | ログイン用パスワードを入力（表示/非表示切替あり） | フォーム中部 |
| ログイン状態維持チェックボックス | 次回自動ログインのオン/オフ | パスワード入力下 |
| ログインボタン | ログイン処理を実行 | フォーム下部 |
| パスワードリセットリンク | パスワードリセット画面へ遷移 | ログインボタン下 |

## 機能とUIの結びつき

| 機能 | UI要素 | インタラクション |
|------|--------|----------------|
| ログイン | メールアドレス入力、パスワード入力、ログインボタン | 入力後ボタンクリックで認証実行、成功時は役割に応じた画面へ遷移 |
| パスワードリセット | パスワードリセットリンク | クリックでモーダル表示、メール入力後に送信 |
| ログイン状態維持 | ログイン状態維持チェックボックス | チェック時は次回アクセス時に自動ログイン |

## 遷移

| 遷移元 | 遷移先 | トリガー |
|--------|--------|----------|
| - | P002 ダッシュボード（教室長） | ログイン成功（教室長の場合） |
| - | P003 マネジメントダッシュボード | ログイン成功（エリアマネジャーの場合） |
| - | P012 ユーザー管理画面 | ログイン成功（システム管理者の場合） |

## 備考

- 認証失敗時はエラーメッセージを表示
- パスワードリセットはモーダルで完結（別画面遷移なし）
