# 仕様書: 講師管理画面

## 基本情報

| 項目 | 内容 |
|-----|------|
| ページID | P005 |
| ページ名 | 講師管理画面 |
| 対象ユーザー | 教室長 |

## 目的

教室に所属する講師の情報を登録・編集・管理する。

## 機能一覧

| 機能ID | 機能 | 説明 |
|--------|------|------|
| F040 | 講師一覧表示 | 登録済み講師の一覧を表示 |
| F041 | 講師新規登録 | 新しい講師を登録 |
| F042 | 講師情報編集 | 既存講師の情報を編集 |
| F043 | 講師削除 | 講師を削除（論理削除） |
| F044 | 講師CSVインポート | CSVファイルから講師を一括登録 |
| F045 | 講師検索・フィルター | 講師名、科目、学年などで絞り込み |

## UI要素

| 要素 | 説明 | 位置 |
|------|------|------|
| ヘッダー | 共通ヘッダー | 上部固定 |
| サイドバー | 共通サイドバー | 左側固定 |
| 検索バー | 講師名で検索 | メインエリア上部左 |
| フィルターボタン | 科目・学年・性別などでフィルター | 検索バー横 |
| 新規登録ボタン | 講師登録モーダルを開く | メインエリア上部右 |
| CSVインポートボタン | CSVアップロードモーダルを開く | 新規登録ボタン横 |
| 講師一覧テーブル | 講師情報の一覧 | メインエリア中部 |
| 編集ボタン | 各行の編集ボタン | テーブル内 |
| 削除ボタン | 各行の削除ボタン | テーブル内 |
| ページネーション | ページ切り替え | テーブル下部 |

## 講師一覧テーブルの表示項目

| 項目 | 説明 |
|------|------|
| 講師ID | システム内の一意ID |
| 氏名 | 講師の氏名 |
| 性別 | 男/女 |
| 指導可能科目 | 主要科目を表示（全件はツールチップ） |
| 指導可能学年 | 小/中/高の範囲で表示 |
| コマ数設定 | 最小〜最大コマ/週 |
| 大学ランク | A/B/C |
| ステータス | 有効/無効 |

## 講師登録・編集モーダルの項目

### 基本情報

| 項目 | 型 | 必須 | 説明 |
|------|-----|-----|------|
| 講師ID | 自動採番 | - | システムが自動付与 |
| 氏名 | テキスト | ○ | 講師の氏名 |
| 性別 | 選択 | ○ | 男/女 |

### 指導能力

| 項目 | 型 | 必須 | 説明 |
|------|-----|-----|------|
| 指導可能科目 | 複数選択 | ○ | 科目マスタから選択（複数可） |
| 指導可能学年 | 複数選択 | ○ | 小1〜高3から選択（複数可） |

### 勤務条件

| 項目 | 型 | 必須 | 説明 |
|------|-----|-----|------|
| 最小コマ/週 | 数値 | ○ | 週あたりの最低勤務コマ数 |
| 最大コマ/週 | 数値 | ○ | 週あたりの最大勤務コマ数 |
| 最大連続コマ | 数値 | ○ | 連続して担当できる最大コマ数 |

### マッチング条件

| 項目 | 型 | 必須 | 説明 |
|------|-----|-----|------|
| NG生徒 | 生徒複数選択 | - | 担当不可の生徒を指定 |
| 中学受験経験 | チェック | - | 中学受験の指導経験・知識あり |
| 高校受験経験 | チェック | - | 高校受験の指導経験・知識あり |
| 大学ランク | 選択 | - | A/B/C（志望レベルとのマッチングに使用） |

## 機能とUIの結びつき

| 機能 | UI要素 | インタラクション |
|------|--------|----------------|
| 講師一覧表示 | 講師一覧テーブル | 一覧表示、ソート可能 |
| 講師新規登録 | 新規登録ボタン | クリックでモーダル表示、入力後保存 |
| 講師情報編集 | 編集ボタン | クリックでモーダル表示、編集後保存 |
| 講師削除 | 削除ボタン | クリックで確認ダイアログ、確定で削除 |
| CSVインポート | CSVインポートボタン | クリックでモーダル表示、ファイル選択後インポート |
| 検索・フィルター | 検索バー、フィルターボタン | 入力・選択で一覧を絞り込み |

## フィルター条件

| フィルター項目 | 選択肢 |
|--------------|--------|
| 性別 | 男/女/すべて |
| 指導可能科目 | 科目マスタから選択 |
| 指導可能学年 | 小学生/中学生/高校生/すべて |
| 大学ランク | A/B/C/すべて |
| 中学受験経験 | あり/なし/すべて |
| 高校受験経験 | あり/なし/すべて |
| ステータス | 有効/無効/すべて |

## 遷移

| 遷移元 | 遷移先 | トリガー |
|--------|--------|----------|
| P002 ダッシュボード | - | サイドバー「講師管理」クリック |
| - | P002 ダッシュボード | サイドバーで他画面選択 |

## 備考

- 削除は論理削除（過去の時間割データとの整合性維持）
- CSVインポート時はプレビュー表示してから確定
- 指導可能科目・学年は科目マスタ・学年マスタから選択
- NG生徒は生徒マスタから選択（相互関係：生徒側のNG講師とも連動）
