# 仕様書: ダッシュボード（教室長）

## 基本情報

| 項目 | 内容 |
|-----|------|
| ページID | P002 |
| ページ名 | ダッシュボード（教室長） |
| 対象ユーザー | 教室長 |

## 目的

教室長がログイン後に最初に目にする画面。自教室の充足率・人員状況をサマリー表示し、各機能への導線を提供する。

## 機能一覧

| 機能ID | 機能 | 説明 |
|--------|------|------|
| F010 | ダッシュボード表示（教室長） | 教室長向けメイン画面を表示 |
| F011 | 充足率サマリー表示 | 自教室の1対2充足率を数値・グラフで表示 |
| F012 | 人員状況ヒートマップ表示 | 講師の余剰・不足状況をヒートマップ形式で表示 |
| F013 | 科目別カバー率表示 | 科目ごとの講師充足状況を表示 |
| F014 | 需給バランス表示 | 生徒の希望コマ数と講師の提供可能コマ数のバランスを表示 |
| F015 | 通知・アラート表示 | 対応が必要な事項（欠席連絡など）を表示 |

## 人員状況ヒートマップの構成

| 軸 | 内容 |
|----|------|
| 横軸（曜日） | 月〜金（土曜営業の場合は土も含む） |
| 縦軸（時間帯） | 1限〜4限（土曜は0限〜4限） |
| セル色 | 緑（余裕あり）、黄緑（適正）、黄（ギリギリ）、赤（不足） |

### 需給差の計算

各時間枠（曜日×限）ごとに以下を計算:

| 項目 | 計算式 |
|------|--------|
| 需給差 | 供給枠数 - 需要人数 |
| 供給枠数 | その枠で勤務可能な講師数（シフト希望○） × 2（1対2形式想定） |
| 需要人数 | その枠で受講可能な生徒数（受講希望○または△） |

### セル色の判定基準

| 需給差 | ステータス | 表示色 |
|--------|-----------|--------|
| 2以上 | 余裕あり | 緑 |
| 1 | 適正 | 黄緑 |
| 0 | ギリギリ | 黄 |
| マイナス | 不足 | 赤 |

## 科目別カバー率の対象科目

実際のデータに基づき、以下のカテゴリで表示:

| カテゴリ | 科目例 |
|---------|--------|
| 小学生 | 小学英語、公立算数、中受算数、小学国語、小論、公立理科、中受理科、公立社会、中受社会 |
| 中学生 | 中学英語、公立数学、私立数学、中学国語、中学理科、中学社会 |
| 高校生 | 高校12英語、高3英語、高校国語、数1A、数ⅡB、数Ⅲ、生物、化学、物理、世界史、日本史、地理 |

### 科目別カバー率の計算

科目ごとに「その科目の授業を十分に提供できる講師がいるか」を示す。

```
科目別カバー率 = min(供給充足度, 時間帯充足度) × 100
```

**供給充足度（按分方式）**:
- 講師が複数科目を教えられる場合、その講師のコマ数を各科目の需要比率で按分
- 例: 講師が数学と英語を教えられる場合、数学需要:英語需要の比率でコマ数を配分
- これにより講師リソースの二重カウントを防ぐ

**時間帯充足度**:
- 各時間枠で「その科目を教えられる講師がいるか」を確認
- 講師と生徒の希望時間帯がマッチしない場合はカバー率が下がる

### カバー率の判定基準

| カバー率 | ステータス | 表示色 |
|---------|-----------|--------|
| 100%以上 | 充足 | 緑 |
| 70〜99% | やや不足 | 黄 |
| 70%未満 | 不足 | 赤 |

## 需給バランスの計算

週全体の講師供給数と生徒需要数をグラフで可視化する。

| 指標 | 算出方法 |
|------|----------|
| 生徒需要 | 全生徒の科目別コマ数の合計 |
| 講師供給 | 各講師の最大コマ/週の合計 |
| 差分 | 供給 - 需要（マイナスは不足、プラスは余剰） |

## 機能とUIの結びつき

| 機能 | UI要素 | インタラクション |
|------|--------|----------------|
| 充足率サマリー表示 | 充足率カード | 表示のみ（クリックで詳細表示も可） |
| 人員状況サマリー表示 | 人員状況ヒートマップ | ヒートマップで視覚的に把握、セルクリックで詳細 |

| 各機能へのナビゲーション | サイドバー、クイックアクションボタン | クリックで各画面へ遷移 |
| 通知・アラート表示 | 通知リスト | クリックで該当画面へ遷移 |
