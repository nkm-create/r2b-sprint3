# 仕様書: 条件設定画面

## 基本情報

| 項目 | 内容 |
|-----|------|
| ページID | P008 |
| ページ名 | 条件設定画面 |
| 対象ユーザー | 教室長 |

## 目的

マスタデータ（講師・生徒・教室設定）から自動反映された制約を確認し、ターム固有の調整と全体ポリシーを設定する。

## 設計思想

### マスタデータとの役割分担

| データソース | 管理場所 | P008での扱い |
|-------------|---------|--------------|
| 講師の基本情報（科目、コマ数、連続上限等） | P005 講師管理 | **自動反映**（参照のみ） |
| 生徒の基本情報（希望講師、NG講師、性別希望等） | P006 生徒管理 | **自動反映**（参照のみ） |
| 教室設定（ブース数、営業曜日等） | P004 教室設定 | **自動反映**（参照のみ） |
| シフト希望・受講希望 | P007 希望データ | **自動反映**（参照のみ） |
| **ターム固有の追加調整** | P008 条件設定 | **編集可能** |
| **全体ポリシー** | P008 条件設定 | **編集可能** |

### P008の役割

1. **マスタから自動反映された制約の確認**: 講師・生徒・教室設定から読み込み、プリセット表示
2. **ターム固有の追加調整**: マスタ設定に対する「このターム限定の上書き/追加」
3. **全体ポリシー設定**: マッチングルール等の全体方針

## ハード制約（常時適用・変更不可）

以下のハード制約は自動適用され、ユーザーによる編集・削除はできない。

| 制約ID | 制約名 | 説明 | データソース |
|--------|--------|------|------------|
| H001 | 科目適合 | 講師の指導可能科目に生徒の受講科目が含まれること | 講師マスタ、生徒マスタ |
| H002 | 学年適合 | 講師の指導可能学年に生徒の学年が含まれること | 講師マスタ、生徒マスタ |
| H003 | 講師出勤可能時間 | シフト希望で○の時間帯のみ配置 | 講師シフト希望 |
| H004 | 生徒通塾可能時間 | 受講希望で○/△の時間帯のみ配置 | 生徒受講希望 |
| H005 | NG組み合わせ禁止 | NG講師/NG生徒の組み合わせは配置しない | 講師マスタ、生徒マスタ |

## マスタ由来ソフト制約（自動反映・ターム調整可）

マスタデータから自動的にプリセットされる制約。ターム固有の調整が可能。

| 制約ID | 制約名 | データソース | ターム調整例 |
|--------|--------|------------|-------------|
| S001 | 講師コマ数範囲 | 講師マスタ.最小コマ/週、最大コマ/週 | 「今月は田中先生を最大3コマに」 |
| S002 | 講師連続コマ上限 | 講師マスタ.最大連続コマ | 「今月は山本先生を連続2コマまでに」 |
| S003 | 生徒連続コマ上限 | 生徒マスタ.最大連続コマ | 「今月は鈴木くんを連続1コマまでに」 |
| S004 | 講師希望 | 生徒マスタ.希望講師ID | 「今月は追加で佐藤先生を希望」 |
| S005 | 講師NG | 生徒マスタ.NG講師ID | 「今月は追加で田中先生をNG」 |
| S006 | 講師性別希望 | 生徒マスタ.講師希望性別 | （通常は調整不要） |
| S007 | ブースキャパシティ | 教室設定.ブース数 | 「今月は改装で最大6コマに」 |

## 全体ポリシー（P008で設定）

マッチング・最適化の全体方針。教室全体に適用。

| ポリシーID | ポリシー名 | 説明 |
|-----------|-----------|------|
| P001 | 志望レベル→大学ランクマッピング | 生徒の志望レベルに応じて講師ランクを制限 |
| P002 | 中受経験優先 | 中受科目に中受経験講師を優先配置 |
| P003 | 高受経験優先 | 高受科目に高受経験講師を優先配置 |
| P004 | 同一科目ペアリング | 1対2で同一科目の生徒同士を優先 |
| P005 | 充足率目標 | 1対2コマの充足率目標値（50%〜100%） |
| P006 | 時間帯優先 | 特定時間帯を優先的に埋める |

## 備考

- **再入力不要**: マスタデータから自動的にプリセットされるため、ユーザーは差分のみを設定
- **ハード制約とソフト制約の区別**: ハード制約（H001〜H005）は常時適用・編集不可。ソフト制約はターム調整可能
- **ターム調整の用途**: 試験期間でコマを減らしたい、一時的にNG講師を追加したい等のターム限定ニーズに対応
- **テンプレート機能**: よく使う全体ポリシー設定をテンプレートとして保存・再利用可能
- **マスタへのリンク**: 恒久的な設定変更が必要な場合は各マスタ管理画面へ遷移可能
