# 仕様書: マネジメントダッシュボード

## 基本情報

| 項目 | 内容 |
|-----|------|
| ページID | P003 |
| ページ名 | マネジメントダッシュボード |
| 対象ユーザー | エリアマネジャー |

## 目的

エリアマネジャーが担当エリア内の複数教室の充足率・人員状況を一覧で把握し、問題のある教室を特定できるようにする。

## 機能一覧

| 機能ID | 機能 | 説明 |
|--------|------|------|
| F020 | マネジメントダッシュボード表示 | エリアマネジャー向けメイン画面を表示 |
| F021 | 教室別充足率一覧表示 | 各教室の充足率をランキング・一覧形式で表示 |
| F022 | 教室詳細ドリルダウン | 特定教室の詳細情報をパネル表示 |
| F023 | 問題教室アラート | 充足率が低い、人員に問題がある教室をアラート表示 |
| F024 | 教室比較 | 複数教室の指標を並べて比較 |

## UI要素

| 要素 | 説明 | 位置 |
|------|------|------|
| ヘッダー | ロゴ、エリア名、ユーザー名、ログアウトボタン | 上部固定 |
| 期間セレクター | 表示対象タームを選択 | ヘッダー下 |
| エリアサマリーカード | エリア全体の充足率・人員状況 | メインエリア上部 |
| 教室一覧テーブル | 教室名、充足率、人員状況、アラート有無 | メインエリア中部 |
| 教室詳細パネル | 選択した教室の詳細情報（スライドイン） | 右側パネル |
| アラートバッジ | 問題のある教室に付与 | テーブル内 |
| 比較ボタン | 選択した複数教室を比較表示 | テーブル上部 |

## 教室一覧テーブルの表示項目

| 項目 | 説明 |
|------|------|
| 教室名 | 教室の名称 |
| 充足率 | 1対2充足率（%） |
| ステータス | 正常/警告/危険（色分け） |
| 前月比 | 充足率の増減（↑/↓/→） |
| 講師数 | 有効な講師の人数 |
| 生徒数 | 有効な生徒の人数 |
| 時間割ステータス | 確定/ドラフト/未作成 |
| アラート | 問題がある場合にバッジ表示 |

## 教室詳細パネルの表示内容

P002ダッシュボード（教室長）と同等の情報を参照モードで表示。

| セクション | 表示内容 |
|-----------|---------|
| サマリー | 充足率、講師数、生徒数 |
| 充足率推移グラフ | 過去6ターム分の推移 |
| 人員ヒートマップ | 曜日×時限の講師/生徒配置状況 |
| 時間割サマリー | 現在の時間割概要 |
| 問題点リスト | 充足率低下要因等の注意事項 |

## 機能とUIの結びつき

| 機能 | UI要素 | インタラクション |
|------|--------|----------------|
| エリア全体サマリー表示 | エリアサマリーカード | 表示のみ |
| 教室別充足率一覧 | 教室一覧テーブル | 充足率でソート可能、フィルター可能 |
| 教室別人員状況一覧 | 教室一覧テーブル | 人員状況列で確認 |
| 問題教室アラート | アラートバッジ | 視覚的に問題教室を識別、上部に固定表示 |
| 教室詳細ドリルダウン | 教室詳細パネル | テーブル行クリックで右パネルにスライドイン表示 |
| 教室比較 | 比較ボタン、比較モーダル | チェックボックスで複数選択後、比較ボタンで比較表示 |
| 期間フィルター | 期間セレクター | 選択で表示データを切り替え |

## 遷移

| 遷移元 | 遷移先 | トリガー |
|--------|--------|----------|
| P001 ログイン画面 | - | ログイン成功（エリアマネジャーの場合） |
| - | 教室詳細パネル | 教室行クリック |
| 教室詳細パネル | 一覧に戻る | ×ボタンクリック、パネル外クリック |

## ドリルダウンの詳細

### 教室詳細パネルの動作

1. 教室一覧テーブルの行をクリック
2. 右側から教室詳細パネルがスライドイン
3. P002ダッシュボード相当の情報を参照モードで表示
4. ×ボタンまたはパネル外クリックで閉じる

### 参照モードの制約

- 表示のみ、編集操作は不可
- 時間割の調整、マスタの編集等はできない
- 問題の特定と状況把握のみに使用

## アラート条件

| 条件 | アラートレベル | 表示 |
|------|--------------|------|
| 充足率60%未満 | 危険 | 赤バッジ |
| 前月比10ポイント以上低下 | 危険 | 赤バッジ |
| 充足率60〜79% | 警告 | 黄バッジ |
| 未確定の時間割が期限超過 | 警告 | 黄バッジ |

## 備考

- 教室詳細は別画面に遷移せず、右パネルにスライドイン表示
- 詳細パネルでは充足率推移グラフ、人員ヒートマップ、時間割サマリーを表示
- エリアマネジャーは閲覧のみ、編集機能は持たない
- アラートのある教室は一覧上部に固定表示（優先的に確認できるよう）
- 複数教室の比較機能で、教室間の状況を並べて確認可能
