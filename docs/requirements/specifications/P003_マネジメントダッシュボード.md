# 仕様書: マネジメントダッシュボード

## 基本情報

| 項目 | 内容 |
|-----|------|
| ページID | P003 |
| ページ名 | マネジメントダッシュボード |
| 対象ユーザー | エリアマネジャー |

## 目的

エリアマネジャーが担当エリア内の複数教室の充足率・人員状況を一覧で把握し、問題のある教室を特定できるようにする。

## 機能一覧

| 機能ID | 機能 | 説明 |
|--------|------|------|
| F020 | マネジメントダッシュボード表示 | エリアマネジャー向けメイン画面を表示 |
| F021 | 教室別充足率一覧表示 | 各教室の充足率をランキング・一覧形式で表示 |
| F022 | 教室詳細ドリルダウン | 特定教室の詳細情報をパネル表示 |
| F023 | 問題教室アラート | 充足率が低い、人員に問題がある教室をアラート表示 |

## UI要素

| 要素 | 説明 | 位置 |
|------|------|------|
| ヘッダー | ロゴ、エリア名、ユーザー名、ログアウトボタン | 上部固定 |
| 期間セレクター | 表示対象タームを選択 | ヘッダー下 |
| エリアサマリーカード | エリア全体の充足率・人員状況 | メインエリア上部 |
| 教室一覧テーブル | 教室名、充足率、人員状況、アラート有無 | メインエリア中部 |
| 教室詳細パネル | 選択した教室の詳細情報（スライドイン） | 右側パネル |
| アラートバッジ | 問題のある教室に付与 | テーブル内 |

## 機能とUIの結びつき

| 機能 | UI要素 | インタラクション |
|------|--------|----------------|
| エリア全体サマリー表示 | エリアサマリーカード | 表示のみ |
| 教室別充足率一覧 | 教室一覧テーブル | 充足率でソート可能 |
| 教室別人員状況一覧 | 教室一覧テーブル | 人員状況列で確認 |
| 問題教室アラート | アラートバッジ | 視覚的に問題教室を識別 |
| 教室詳細ドリルダウン | 教室詳細パネル | テーブル行クリックで右パネルに詳細表示 |
| 期間フィルター | 期間セレクター | 選択で表示データを切り替え |

## 遷移

| 遷移元 | 遷移先 | トリガー |
|--------|--------|----------|
| P001 ログイン画面 | - | ログイン成功（エリアマネジャーの場合） |

## 備考

- 教室詳細は別画面に遷移せず、右パネルにスライドイン表示
- 詳細パネルでは充足率推移グラフ、人員ヒートマップ、時間割サマリーを表示
- エリアマネジャーは閲覧のみ、編集機能は持たない
