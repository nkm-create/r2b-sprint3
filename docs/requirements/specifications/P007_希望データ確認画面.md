# 仕様書: 希望データ確認画面

## 基本情報

| 項目 | 内容 |
|-----|------|
| ページID | P007 |
| ページ名 | 希望データ確認画面 |
| 対象ユーザー | 教室長 |

## 目的

Google Formから取り込んだ講師・生徒の希望データを確認・編集する。

## 機能一覧

| 機能ID | 機能 | 説明 |
|--------|------|------|
| F060 | 希望データマトリクス表示 | 講師・生徒の希望データをマトリクス形式で表示 |
| F061 | 講師/生徒希望切り替え | 講師希望 / 生徒希望の表示切り替え |
| F062 | 希望データ編集 | セルをクリックして希望値を編集（○/×/△） |
| F063 | Google Form同期 | Google Formから最新データを取り込み |
| F064 | 未回答者確認 | まだ希望を提出していない講師・生徒を表示 |

## UI要素

| 要素 | 説明 | 位置 |
|------|------|------|
| ヘッダー | 共通ヘッダー | 上部固定 |
| サイドバー | 共通サイドバー | 左側固定 |
| データ種別タブ | 講師希望 / 生徒希望 / 未回答者の切り替え | メインエリア上部 |
| 同期ボタン | Google Formからデータ取り込み | タブ横 |
| 最終同期日時 | 前回の同期日時を表示 | 同期ボタン横 |
| 希望マトリクス | 縦軸: 講師/生徒名、横軸: 曜日×限の希望データ | メインエリア中部 |
| セル値 | ○（希望）/ ×（不可）/ △（できれば可） | マトリクス内 |
| 時間帯サマリー行 | 各時間帯の希望者数合計を表示 | マトリクス下部 |
| 編集モード切替 | 閲覧モード / 編集モードの切り替え | メインエリア上部右 |
| 回答督促ボタン | 未回答者へのリマインド（将来対応） | 未回答者タブ内 |
| 時間割作成ボタン | P008へ遷移 | メインエリア上部右 |

## 希望マトリクスの構成

### 講師シフト希望

| 項目 | 説明 |
|------|------|
| 縦軸 | 講師名（ID順またはアイウエオ順） |
| 横軸 | 曜日×限（月1限、月2限、...、土4限まで最大26コマ） |
| セル値 | ○: 勤務可能、×: 勤務不可 |

#### 時間帯の構成

| 曜日 | 限 |
|------|-----|
| 月〜金 | 1限、2限、3限、4限 |
| 土 | 0限、1限、2限、3限、4限 |

### 生徒受講希望

| 項目 | 説明 |
|------|------|
| 縦軸 | 生徒名（ID順またはアイウエオ順） |
| 横軸 | 曜日×限（月1限、月2限、...、土4限まで最大26コマ） |
| セル値 | ○: 希望、×: 不可、△: できれば可 |

## 希望値の意味

| 値 | 講師での意味 | 生徒での意味 |
|----|-------------|-------------|
| ○ | 勤務可能 | 受講希望 |
| × | 勤務不可 | 受講不可 |
| △ | - | できれば受講可能（調整可） |

## 機能とUIの結びつき

| 機能 | UI要素 | インタラクション |
|------|--------|----------------|
| 希望データ一覧表示 | 希望マトリクス | 曜日×限のマトリクス形式で一覧表示 |
| データ種別切り替え | データ種別タブ | タブクリックで講師/生徒/未回答者を切り替え |
| 希望データ編集 | セル値 | セルクリックで○/×/△をトグル切り替え |
| データ同期 | 同期ボタン | クリックで同期実行、完了後マトリクス更新 |
| 未回答者確認 | 未回答者タブ | タブクリックで未回答者一覧表示 |
| 希望サマリー表示 | 時間帯サマリー行 | 自動計算で各時間帯の希望者数を表示 |

## 遷移

| 遷移元 | 遷移先 | トリガー |
|--------|--------|----------|
| P002 ダッシュボード | - | サイドバー「希望データ」クリック |
| - | P008 条件設定画面 | 「時間割作成へ進む」ボタン |

## 備考

- Google Form連携の設定は別途設定画面または初期セットアップで行う
- 手動編集した場合は編集済みフラグを付与（セルの色を変えて区別）
- 同期時に既存の手動編集データを上書きするかどうかは確認ダイアログで選択
- マトリクスは横スクロール可能（26コマ分を表示）
- 時間帯サマリーでは、各コマの希望者数と必要数の比較を色分けで表示
  - 緑: 希望者が十分
  - 黄: やや不足
  - 赤: 不足
