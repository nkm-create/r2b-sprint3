| 項目 | 内容 |

## 3. 機能詳細

### F100: ターム一覧表示

#### 出力項目

| 項目名 | 説明 |
|--------|------|
| ターム名 | タームの名称（例: 2025年4月期） |
| 期間 | 開始日〜終了日 |
| ステータス | "作成中" / "確定" / "アーカイブ" |
| 充足率 | 1対2充足率 |
| 時間割バージョン数 | 作成されたバージョン数 |

#### ステータス説明

| ステータス | 説明 |
|-----------|------|
| 作成中 | 時間割を作成・調整中 |
| 確定 | 時間割が確定済み |
| アーカイブ | 過去のターム（参照のみ） |

---

### F101: ターム詳細確認

#### 出力項目

| 項目名 | 説明 |
|--------|------|
| ターム基本情報 | 名称、期間、ステータス |
| 時間割バージョン一覧 | 各バージョンの情報 |
| 統計情報 | 充足率、講師数、生徒数、総コマ数 |
| 制約条件サマリー | 設定されている制約の概要 |

#### 時間割バージョン一覧

| 項目名 | 説明 |
|--------|------|
| バージョン番号 | v1, v2, ... |
| ステータス | draft / confirmed / archived |
| 作成日時 | バージョン作成日時 |
| 充足率 | そのバージョンの充足率 |
| 作成者 | バージョンを作成したユーザー |

---

### F102: ターム複製

元タームの条件設定をコピーして新ターム作成。

#### 処理

1. 複製元タームを選択
2. 新ターム名・期間を入力
3. コピーする項目を選択（制約条件、マスタスナップショット等）
4. 複製実行
5. 条件設定画面（P008）へ遷移

#### コピー対象選択

| 項目 | デフォルト | 説明 |
|------|----------|------|
| 制約条件 | ○ | ソフト制約のコピー |
| 講師マスタ | × | 講師データのスナップショット |
| 生徒マスタ | × | 生徒データのスナップショット |
| 希望データ | × | シフト・受講希望のコピー |

---

### F103: ターム編集

#### 入力項目

| 項目名 | 型 | 必須 | バリデーション |
|--------|-----|------|--------------|
| ターム名 | string | ○ | 最大50文字、同一教室内で重複不可 |
| 開始日 | date | ○ | 終了日より前 |
| 終了日 | date | ○ | 開始日より後 |

#### 制約

- 確定済み時間割がある場合、期間の変更は警告表示

#### 期間重複のルール

**同一教室内でのターム期間の重複は許可する。**

| 状況 | 許可 | 理由 |
|------|-----|------|
| 期間が完全に同じ | ○ | 複数パターンの時間割を比較検討する用途 |
| 期間が部分的に重複 | ○ | 移行期間で両方のタームを参照する用途 |
| 同一ターム名 | × | 混乱を避けるためターム名は一意とする |

**重複時の表示**:
- ターム一覧画面で期間が重複するタームには「重複あり」のラベルを表示
- 重複するターム同士をグループ表示（同じ背景色でハイライト）

**運用上の注意**:
- 「確定」ステータスにできるのは、同一期間内で1タームのみ
- 他のタームが確定済みの期間に新タームを確定しようとした場合、警告を表示

---

### F104: ターム削除

論理削除。確定済み時間割がある場合はアーカイブ化。

#### 処理

| 条件 | 動作 |
|------|------|
| 時間割なし | 完全削除（論理削除） |
| draft時間割のみ | 確認後削除 |
| confirmed時間割あり | アーカイブ化（削除不可） |

## 4. UI状態遷移

```
[ターム管理画面]
    ├─→ 新規作成ボタン → [作成モーダル]
    │                       ├─→ 保存 → [条件設定画面へ]
    │                       └─→ キャンセル → [モーダル閉じる]
    ├─→ 行クリック → [詳細画面]
    │                 ├─→ 編集 → [編集モーダル]
    │                 ├─→ 複製 → [複製モーダル] → [条件設定画面へ]
    │                 ├─→ 削除 → [確認ダイアログ]
    │                 │           ├─→ 削除可能 → [削除実行]
    │                 │           └─→ 削除不可 → [アーカイブ化提案]
    │                 └─→ バージョン選択 → [時間割作成画面へ]
    └─→ フィルター/ソート → [一覧更新]
```
