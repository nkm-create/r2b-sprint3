# P010: 振替対応画面 要件定義書 v1

## 1. 画面概要

| 項目 | 内容 |
|------|------|
| 画面ID | P010 |
| 画面名 | 振替対応画面 |
| 対象ユーザー | 教室長 |
| 目的 | 講師・生徒の欠席登録と振替の手配 |

## 2. 機能一覧

| ID | 機能名 | 優先度 |
|----|--------|--------|
| F090 | 欠席登録 | Must |
| F091 | 振替候補提案 | Must |
| F092 | 振替確定 | Must |
| F093 | 欠席・振替履歴表示 | Must |

---

## 3. 機能詳細

### F090: 欠席登録

#### 入力項目

| 項目名 | 型 | 必須 | 説明 |
|--------|-----|------|------|
| 対象者種別 | enum | ○ | "講師" / "生徒" |
| 対象者 | string | ○ | 講師/生徒マスタから選択 |
| 欠席コマ | object | ○ | 確定済み時間割から選択 |
| 欠席理由 | string | - | 最大200文字 |
| 振替希望有無 | boolean | ○ | デフォルト: true |

#### 処理

1. 対象者の確定済みコマを表示
2. 欠席するコマを選択
3. 欠席理由を入力（任意）
4. 振替希望の有無を選択
5. 登録実行

---

### F091: 振替候補提案

#### 処理フロー

1. 欠席パターン判定（生徒欠席 or 講師欠席）
2. 関係者の希望時間帯を取得（P007のマトリクスデータ参照）
3. 候補スロット検索（空き×希望×制約チェック）
4. 優先度スコア計算
5. 上位10件を返却

#### 欠席パターン別処理

| パターン | 振替方法 |
|---------|----------|
| 生徒欠席 | 同じ講師の別時間帯を探す |
| 講師欠席 | 別の講師（または同じ講師の別日）を探す |

#### 優先度スコア計算

| 要素 | スコア |
|------|--------|
| 元講師維持 | +50 |
| 生徒希望○ | +30 |
| 生徒希望△ | +10 |
| 日程近さ（同週内） | +20 |
| 日程近さ（翌週） | +10 |

#### 出力

| 項目名 | 説明 |
|--------|------|
| slot | 振替先の時間枠 |
| type | RESCHEDULE（時間変更）/ SUBSTITUTE（代講） |
| teacher | 担当講師 |
| priority_score | 優先度スコア |
| reasons | スコアの内訳 |

---

### F092: 振替確定

#### 処理

1. 元コマを「欠席」ステータスに更新
2. 振替先コマを追加
3. 履歴に記録
4. 関係者への通知（任意）

#### 出力

- 振替確定メッセージ
- 更新後の時間割表示

---

### F093: 欠席・振替履歴表示

#### 出力項目

| 項目 | 説明 |
|------|------|
| 登録日時 | 欠席登録日時 |
| 対象者 | 欠席した講師/生徒 |
| 元コマ | 欠席したコマ情報（曜日・時限・科目） |
| 振替先 | 振替先のコマ情報 |
| ステータス | 振替待ち / 振替済み / 振替なし |

#### フィルター

| フィルター | 選択肢 |
|-----------|--------|
| 対象者種別 | 講師/生徒/すべて |
| ステータス | 振替待ち/振替済み/振替なし/すべて |
| 期間 | 日付範囲指定 |

## 4. UI状態遷移

```
[振替対応画面]
    ├─→ 欠席登録タブ
    │     ├─→ 対象者選択 → [コマ一覧表示]
    │     └─→ コマ選択 → [欠席登録フォーム]
    │                      ├─→ 登録（振替希望あり） → [振替候補提案]
    │                      │                          ├─→ 候補選択 → [確認] → [確定]
    │                      │                          └─→ 後で決める → [振替待ちとして保存]
    │                      └─→ 登録（振替希望なし） → [欠席のみ登録]
    └─→ 履歴タブ
          ├─→ 一覧表示 → [F093]
          ├─→ 行クリック（振替待ち） → [振替候補提案]
          └─→ フィルター → [一覧更新]
```
