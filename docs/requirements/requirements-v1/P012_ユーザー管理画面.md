## 3. 機能詳細

### F110: ユーザー一覧表示

ユーザーの一覧をテーブル表示。検索・フィルター対応。

#### 出力項目

| 項目名 | 説明 |
|--------|------|
| 氏名 | ユーザーの氏名 |
| メールアドレス | ログイン用メールアドレス |
| 役割 | 教室長/エリアマネジャー/システム管理者 |
| 担当教室/エリア | 担当範囲 |
| ステータス | 有効/無効 |
| 最終ログイン | 最後にログインした日時 |

---

### F111: ユーザー新規作成

#### 入力項目

| 項目名 | 型 | 必須 | バリデーション |
|--------|-----|------|--------------|
| 氏名 | string | ○ | 最大50文字 |
| メールアドレス | string | ○ | メール形式、重複不可 |
| 役割 | enum | ○ | "教室長" / "エリアマネジャー" / "システム管理者" |
| 担当教室 | string[] | △ | 教室長・エリアマネジャーの場合必須 |
| ステータス | enum | ○ | "有効" / "無効" |

#### 処理

バリデーション → 初期パスワード生成 → 登録 → メール送信

#### 初期パスワード

- 自動生成（英数字混合、12文字）
- 初回ログイン時に変更必須

---

### F112: ユーザー情報編集

F111と同じ項目を編集。

#### 制約

- 自分自身の役割は変更不可
- 最後のシステム管理者は削除/無効化不可

---

### F113: ユーザー削除

論理削除。他データで参照されている場合は無効化のみ。

#### 処理

| 条件 | 動作 |
|------|------|
| 参照データなし | 論理削除 |
| 参照データあり | 無効化のみ（削除不可） |
| 最後の管理者 | 削除不可（エラー） |

---

### F114: パスワードリセット（管理者）

対象ユーザーのパスワードをリセットし、新パスワードをメール送信。

#### 処理

1. 対象ユーザーを選択
2. リセット確認ダイアログ
3. 新パスワード生成
4. パスワード更新
5. メール送信

---

## 4. UI状態遷移

```
[ユーザー管理画面]
    ├─→ 新規作成ボタン → [作成モーダル]
    │                       ├─→ 保存 → [メール送信] → [一覧更新]
    │                       └─→ キャンセル → [モーダル閉じる]
    ├─→ 行クリック → [詳細/編集モーダル]
    │                   ├─→ 編集 → [編集モード]
    │                   │           ├─→ 保存 → [一覧更新]
    │                   │           └─→ キャンセル → [詳細モード]
    │                   ├─→ パスワードリセット → [確認] → [リセット実行]
    │                   └─→ 削除 → [確認ダイアログ]
    │                               ├─→ 削除可能 → [削除実行]
    │                               └─→ 削除不可 → [無効化提案]
    └─→ 検索/フィルター → [一覧更新]
```
