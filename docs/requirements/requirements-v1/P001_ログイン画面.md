## 3. 機能詳細

### F001: ログイン

#### 入力項目

| 項目名 | 型 | 必須 | バリデーション |
|--------|-----|------|--------------|
| メールアドレス | string | ○ | メール形式 |
| パスワード | string | ○ | 最小8文字 |

#### 処理

メール検索 → パスワードハッシュ比較 → JWT発行 → 役割別画面へリダイレクト

#### 出力

- 認証成功: JWTトークン発行、ダッシュボードへリダイレクト
- 認証失敗: エラーメッセージ表示

#### エラーハンドリング

| エラー条件 | メッセージ |
|-----------|----------|
| 認証失敗 | メールアドレスまたはパスワードが正しくありません |
| アカウント無効 | このアカウントは無効化されています |
| ロック | アカウントがロックされました。管理者にお問い合わせください |

---

### F002: パスワードリセット

#### 入力項目

| 項目名 | 型 | 必須 | バリデーション |
|--------|-----|------|--------------|
| メールアドレス | string | ○ | メール形式、登録済み |

#### 処理

リセットトークン生成（有効期限24時間）→ メール送信 → 新パスワード設定

#### 出力

- 成功: 「パスワードリセットメールを送信しました」
- 失敗: エラーメッセージ

---

### F003: ログイン状態維持

#### 処理

JWTリフレッシュトークンによるセッション維持（有効期限: 7日間）

#### 仕様

- アクセストークン有効期限: 1時間
- リフレッシュトークン有効期限: 7日間
- トークンリフレッシュ: アクセストークン期限切れ時に自動実行

## 4. UI状態遷移

```
[初期状態]
    ↓
[ログインフォーム表示]
    ├─→ ログインボタン押下 → [認証処理中] → 成功 → [ダッシュボードへ遷移]
    │                                    └→ 失敗 → [エラー表示]
    └─→ パスワードを忘れた → [パスワードリセットフォーム表示]
                                ├─→ 送信 → [完了メッセージ]
                                └─→ 戻る → [ログインフォーム表示]
```
