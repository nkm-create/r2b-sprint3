## 3. 機能詳細

### F050: 生徒一覧表示

生徒の一覧をテーブル表示。検索・フィルター・ソート対応。

#### 出力項目

| 項目名 | 説明 |
|--------|------|
| 氏名 | 生徒の氏名 |
| 学年 | 小1〜高3 |
| 受講科目 | 科目一覧（省略表示） |
| 総コマ数 | 週あたりの総受講コマ数 |
| 志望レベル | A/B/C |
| ステータス | 有効/無効 |

---

### F051: 生徒新規登録

#### 入力項目

| 項目名 | 型 | 必須 | バリデーション |
|--------|-----|------|--------------|
| 氏名 | string | ○ | 最大50文字 |
| 学年 | enum | ○ | "小1"〜"高3" |
| 科目別コマ数 | object[] | ○ | [{科目, コマ数}]、1件以上 |
| 最大連続コマ | number | ○ | 1〜4 |
| 希望講師 | string | - | 講師マスタから選択 |
| NG講師 | string[] | - | 講師マスタから選択 |
| 講師希望性別 | enum | - | "男" / "女" / "指定なし" |
| 志望レベル | enum | - | "A" / "B" / "C" |
| 通塾目的 | enum | - | "高校受験" / "中学受験" / "内部進学" / "補習" / "その他" |

#### 処理

バリデーション → 学年・科目整合性チェック → 登録 → バージョン作成（v1）

---

### F052: 生徒情報編集

#### 入力項目

| 項目名 | 型 | 必須 | バリデーション |
|--------|-----|------|--------------|
| 氏名 | string | ○ | 最大50文字 |
| 学年 | enum | ○ | "小1"〜"高3" |
| 科目別コマ数 | object[] | ○ | [{科目, コマ数}]、1件以上、学年と科目の整合性チェック |
| 最大連続コマ | number | ○ | 1〜4 |
| 希望講師 | string | - | 講師マスタから選択、有効な講師のみ |
| NG講師 | string[] | - | 講師マスタから選択、希望講師と重複不可 |
| 講師希望性別 | enum | - | "男" / "女" / "指定なし" |
| 志望レベル | enum | - | "A" / "B" / "C" |
| 通塾目的 | enum | - | "高校受験" / "中学受験" / "内部進学" / "補習" / "その他" |

#### 処理

現在の情報表示 → 編集 → バリデーション → 新バージョン作成 → 旧バージョンのis_current=false

---

### F053: 生徒削除

論理削除（deleted_atフラグ）。確定済み時間割で使用中の場合は警告表示。

#### エラーハンドリング

| エラー条件 | 動作 |
|-----------|------|
| 確定済み時間割に含まれる | 警告ダイアログ表示、確認後に削除可能 |
| 振替待ちデータあり | 警告ダイアログ表示、確認後に削除可能 |

---

### F054: 生徒CSVインポート

**制限**: 最大30件/回

#### 入力フォーマット

```csv
氏名,学年,科目1,コマ数1,科目2,コマ数2,最大連続コマ,志望レベル,通塾目的
```

#### 処理

CSVパース → バリデーション → 学年・科目整合性チェック → プレビュー → 一括登録

#### バリデーション

| バリデーション | 内容 |
|--------------|------|
| 氏名 | 必須、最大50文字 |
| 学年 | "小1"〜"高3"のいずれか |
| 科目 | 学年に対応する科目マスタに存在すること |
| コマ数 | 1以上の整数 |

---

### F055: 生徒検索・フィルター

テキスト検索（氏名）、フィルター（学年、科目、志望レベル、通塾目的等）

#### フィルター項目

| フィルター項目 | 選択肢 |
|--------------|--------|
| 学年 | 小学生/中学生/高校生/すべて、または個別学年 |
| 受講科目 | 科目マスタから選択 |
| 志望レベル | A/B/C/すべて |
| 通塾目的 | 高校受験/中学受験/内部進学/補習/その他/すべて |
| ステータス | 有効/無効/すべて |

---

### F056: 生徒CSVエクスポート

登録済み生徒データをCSV形式でエクスポート。

#### 出力フォーマット

```csv
生徒ID,氏名,学年,受講科目,科目別コマ数,最大連続コマ,希望講師,NG講師,講師希望性別,志望レベル,通塾目的,ステータス
```

#### エクスポート範囲

- 全件: 有効な全生徒データ
- フィルター適用後: 現在の検索・フィルター条件に該当するデータのみ

#### ファイル名

`students_YYYYMMDD.csv`

```

    ├─→ 新規登録ボタン → [登録モーダル]
    │                       ├─→ 保存 → [一覧更新]

    │                   ├─→ 編集 → [編集モード]
    │                   │           ├─→ 保存 → [一覧更新]
    │                   │           └─→ キャンセル → [詳細モード]
    │                   └─→ 削除 → [確認ダイアログ] → [削除実行]
    ├─→ CSVインポート → [インポートモーダル]
    │                     ├─→ ファイル選択 → [プレビュー表示]
    │                     ├─→ 実行 → [一括登録]
    │                     └─→ キャンセル → [モーダル閉じる]
```
