- 通知・アラート（F015）

---

### F011: 充足率サマリー表示

#### 計算式

**用語定義**:
- **1対2コマ数**: 1人の講師が2人の生徒を同時に教えるコマの数
- **1対1コマ数**: 1人の講師が1人の生徒を教えるコマの数
- **総生徒コマ数**: 全生徒の受講コマ数の合計

**計算式**:
```
充足率 = (1対2コマで教わる生徒数) / 総生徒コマ数 × 100
       = (1対2コマ数 × 2) / (1対2コマ数 × 2 + 1対1コマ数) × 100
```

**計算例**:
| 項目 | 値 |
|------|-----|
| 1対2コマ数 | 40コマ |
| 1対1コマ数 | 20コマ |
| 1対2で教わる生徒数 | 40 × 2 = 80人コマ |
| 総生徒コマ数 | 80 + 20 = 100人コマ |
| **充足率** | 80 / 100 × 100 = **80%** |

**エッジケースの扱い**:
| 状況 | 充足率 |
|------|--------|
| 生徒が0人（分母が0） | 表示: 「-」（計算対象外） |
| 全員1対2で配置 | 100% |
| 全員1対1で配置 | 0% |
| 未配置の生徒がいる | 未配置コマは分母・分子ともに含めない |

#### 出力

- 充足率のパーセンテージ
- 前週/前月との比較（増減表示）

### F012: 人員状況ヒートマップ表示

曜日×限のマトリクスで需給状況を色分け表示。

#### 需給差の定義

各時間枠（曜日×限）における「講師が提供可能な授業枠数」と「生徒の受講希望人数」のバランスを示す指標。

| 項目 | 計算式 |
|------|--------|
| 需給差 | 供給枠数 - 需要人数 |
| 供給枠数 | その枠で勤務可能な講師数 × 2（1対2形式想定） |
| 需要人数 | その枠で受講可能（○または△）の生徒数 |

**解釈**: 「その枠であと何人の生徒を受け入れ可能か」を示す。

#### 表示色

| 需給差 | ステータス | 表示色 |
|--------|-----------|--------|
| >= 2 | 余裕あり | 緑 |
| 1 | 適正 | 黄緑 |
| 0 | ギリギリ | 黄 |
| < 0 | 不足 | 赤 |

#### データソース

- 供給: 講師シフト希望（◯の講師数）
- 需要: 生徒受講希望（○/△の生徒数）

### F013: 科目別カバー率表示

科目ごとに「その科目の授業を十分に提供できる講師がいるか」を示す指標。

#### この指標で知りたいこと

「数学を週40コマ分教えてほしい生徒がいるけど、数学を教えられる講師は足りている？」という問いに答える。

#### 計算の考え方

##### ステップ1: 需要を集計する

生徒マスタの「科目別コマ数」から、科目ごとの需要コマ数を合計する。

**例**:
| 生徒 | 科目別コマ数 |
|------|-------------|
| 生徒A | 数学:2 |
| 生徒B | 数学:1, 英語:2 |
| 生徒C | 英語:3 |

→ 数学の需要 = 2 + 1 = **3コマ**
→ 英語の需要 = 2 + 3 = **5コマ**

##### ステップ2: 供給を集計する（按分方式）

講師マスタの「最大コマ/週」から供給を計算する。ただし、1人の講師が複数科目を教えられる場合、そのまま足すと二重カウントになる問題がある。

**按分方式（正しい計算）**:

講師Xの4コマを、数学と英語の需要の比率で分ける。

- 数学の需要 = 3コマ
- 英語の需要 = 5コマ
- 合計 = 8コマ

講師Xの供給の配分:
- 数学への供給 = 4 × (3 ÷ 8) = **1.5コマ**
- 英語への供給 = 4 × (5 ÷ 8) = **2.5コマ**

##### ステップ3: カバー率を計算する

```
科目別カバー率 = 供給コマ数 ÷ 需要コマ数 × 100
```

##### ステップ4: 時間帯の制約も確認する

各時間枠ごとに「その科目を教えられる講師がいるか」も確認し、時間帯が合わない場合はカバー率を下げる。

#### 計算式（まとめ）

```
科目別カバー率 = min(供給充足度, 時間帯充足度) × 100

供給充足度 = 按分した供給コマ数 ÷ 需要コマ数

時間帯充足度 =
  全枠の「その科目を教えられる講師数 × 2」の合計 ÷
  全枠の「その科目を受講希望する生徒数」の合計
```

#### 表示

| カバー率 | ステータス | 表示色 |
|---------|-----------|--------|
| 100%以上 | 充足 | 緑 |
| 70〜99% | やや不足 | 黄 |
| 70%未満 | 不足 | 赤 |

#### データソース

- 需要: 生徒マスタの「科目別コマ数」
- 供給: 講師マスタの「指導可能科目」「最大コマ/週」
- 時間帯: 講師シフト希望、生徒受講希望

### F014: 需給バランス表示

週全体の講師供給数と生徒需要数をグラフで可視化

#### 出力

- 棒グラフまたは折れ線グラフ
- 曜日別の需給バランス

---

### F015: 通知・アラート表示

#### アラート条件

- 充足率が閾値以下
- 未回答の希望データあり
- 確定期限が近いターム

#### 出力

- アラート一覧（重要度でソート）
- 未読/既読の状態管理
