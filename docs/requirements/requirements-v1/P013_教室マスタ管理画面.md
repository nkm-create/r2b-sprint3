## 3. 機能詳細

### F120: 教室一覧表示

教室の一覧をテーブル表示。

#### 出力項目

| 項目名 | 説明 |
|--------|------|
| 教室名 | 教室の名称 |
| 教室コード | 一意の識別コード |
| 所属エリア | エリア名 |
| 住所 | 教室の住所 |
| 電話番号 | 連絡先 |
| ステータス | 運営中/閉鎖 |
| 教室長 | 担当教室長 |

---

### F121: 教室新規作成

#### 入力項目

| 項目名 | 型 | 必須 | バリデーション |
|--------|-----|------|--------------|
| 教室名 | string | ○ | 最大100文字 |
| 教室コード | string | ○ | 英数字、重複不可 |
| 所属エリア | string | ○ | エリアマスタから選択 |
| 住所 | string | - | 最大200文字 |
| 電話番号 | string | - | 電話番号形式 |
| ステータス | enum | ○ | "運営中" / "閉鎖" |

#### 処理

バリデーション → 登録 → 教室設定のデフォルト値適用

---

### F122: 教室情報編集

F121と同じ項目を編集。

#### 制約

- 教室コードは変更不可（読み取り専用）
- 講師・生徒データがある教室はステータスを「閉鎖」に変更時に警告

---

### F123: 教室削除

論理削除。講師・生徒データがある場合は警告表示。

#### 処理

| 条件 | 動作 |
|------|------|
| データなし | 論理削除 |
| 講師・生徒データあり | 警告表示、確認後に削除可能 |
| 確定済み時間割あり | 警告表示（データ移行推奨）、確認後に削除可能 |

---

## 4. UI状態遷移

```
[教室マスタ管理画面]
    ├─→ 新規作成ボタン → [作成モーダル]
    │                       ├─→ 保存 → [一覧更新]
    │                       └─→ キャンセル → [モーダル閉じる]
    ├─→ 行クリック → [詳細/編集モーダル]
    │                   ├─→ 編集 → [編集モード]
    │                   │           ├─→ 保存 → [一覧更新]
    │                   │           └─→ キャンセル → [詳細モード]
    │                   └─→ 削除 → [確認ダイアログ]
    │                               ├─→ データなし → [削除実行]
    │                               └─→ データあり → [警告表示] → [確認後削除]
    └─→ 検索/フィルター → [一覧更新]
```
