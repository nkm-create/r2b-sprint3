## 3. 機能詳細

### F040: 講師一覧表示

講師の一覧をテーブル表示。検索・フィルター・ソート対応。

#### 出力項目

| 項目名 | 説明 |
|--------|------|
| 氏名 | 講師の氏名 |
| 性別 | 男/女 |
| 指導可能科目 | 科目一覧（省略表示） |
| 指導可能学年 | 学年範囲 |
| 最小/最大コマ | 週あたりのコマ数 |
| ステータス | 有効/無効 |

---

### F041: 講師新規登録

#### 入力項目

| 項目名 | 型 | 必須 | バリデーション |
|--------|-----|------|--------------|
| 氏名 | string | ○ | 最大50文字 |
| 性別 | enum | ○ | "男" / "女" |
| 指導可能科目 | string[] | ○ | 科目マスタから選択、1件以上 |
| 指導可能学年 | string[] | ○ | "小1"〜"高3"から選択、1件以上 |
| 最小コマ/週 | number | ○ | 1以上、最大コマ以下 |
| 最大コマ/週 | number | ○ | 最小コマ以上 |
| 最大連続コマ | number | ○ | 1〜4 |
| NG生徒 | string[] | - | 生徒マスタから選択 |
| 中学受験経験 | boolean | - | デフォルト: false |
| 高校受験経験 | boolean | - | デフォルト: false |
| 大学ランク | enum | - | "A" / "B" / "C" / null |

#### 処理

バリデーション → 登録 → バージョン作成（v1）

---

### F042: 講師情報編集

F041と同じ項目を編集。変更時は新バージョンを作成。

#### 処理

現在の情報表示 → 編集 → バリデーション → 新バージョン作成 → 旧バージョンのis_current=false

---

### F043: 講師削除

論理削除（deleted_atフラグ）。確定済み時間割で使用中の場合は警告表示。

#### エラーハンドリング

| 条件 | 動作 |
|------|------|
| 確定済み時間割に含まれる | 警告ダイアログ表示、確認後に削除可能 |

---

### F044: 講師CSVインポート

**制限**: 最大30件/回

#### 入力フォーマット

```csv
氏名,性別,指導可能科目,指導可能学年,最小コマ,最大コマ,最大連続コマ,大学ランク,中受経験,高受経験
```

#### 処理

CSVパース → バリデーション → プレビュー → 一括登録

---

### F045: 講師検索・フィルター

テキスト検索（氏名）、フィルター（科目、学年、性別等）

#### フィルター項目

| フィルター | 選択肢 |
|-----------|--------|
| 性別 | 男/女/すべて |
| 指導可能科目 | 科目マスタから選択 |
| 指導可能学年 | 小学生/中学生/高校生/すべて |
| 大学ランク | A/B/C/すべて |
| ステータス | 有効/無効/すべて |

---

### F046: 講師CSVエクスポート

登録済み講師データをCSV形式でエクスポート。

#### 出力フォーマット

```csv
講師ID,氏名,性別,指導可能科目,指導可能学年,最小コマ,最大コマ,最大連続コマ,大学ランク,中受経験,高受経験,ステータス
```

#### エクスポート範囲

- 全件: 有効な全講師データ
- フィルター適用後: 現在の検索・フィルター条件に該当するデータのみ

#### ファイル名

`teachers_YYYYMMDD.csv`

## 4. UI状態遷移

```
[講師一覧画面]
    ├─→ 新規登録ボタン → [登録モーダル]
    │                       ├─→ 保存 → [一覧更新]
    │                       └─→ キャンセル → [モーダル閉じる]
    ├─→ 行クリック → [詳細/編集モーダル]
    │                   ├─→ 編集 → [編集モード]
    │                   │           ├─→ 保存 → [一覧更新]
    │                   │           └─→ キャンセル → [詳細モード]
    │                   └─→ 削除 → [確認ダイアログ] → [削除実行]
    ├─→ CSVインポート → [インポートモーダル]
    │                     ├─→ ファイル選択 → [プレビュー表示]
    │                     ├─→ 実行 → [一括登録]
    │                     └─→ キャンセル → [モーダル閉じる]
    ├─→ CSVエクスポート → [範囲選択] → [ダウンロード]
    └─→ 検索/フィルター → [一覧更新]
```
