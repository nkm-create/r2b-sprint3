## 3. 機能詳細

### F020: マネジメントダッシュボード表示

エリアマネジャー向け。担当エリア内の全教室サマリーを表示。

#### 入力

- ログインユーザーの担当エリア情報

#### 出力

- エリア全体のサマリー（平均充足率、問題教室数）
- 教室別一覧（F021）
- 問題教室アラート（F023）

---

### F021: 教室別充足率一覧表示

#### 出力項目

| 項目名 | 説明 |
|--------|------|
| 教室名 | 教室の名称 |
| 充足率 | 1対2充足率（%） |
| ステータス | 正常/警告/危険 |
| 前月比 | 充足率の増減 |
| 講師数 | 有効な講師の人数 |
| 生徒数 | 有効な生徒の人数 |
| 時間割ステータス | 確定/ドラフト/未作成 |
| アラート | 問題がある場合にバッジ表示 |

#### ステータス判定基準

| 充足率 | ステータス | 表示色 |
|--------|-----------|--------|
| 80%以上 | 正常 | 緑 |
| 60〜79% | 警告 | 黄 |
| 60%未満 | 危険 | 赤 |

#### ソート・フィルター

| 操作 | 説明 |
|------|------|
| 充足率ソート | 昇順/降順で並び替え |
| ステータスフィルター | 正常/警告/危険で絞り込み |
| アラートフィルター | アラートありのみ表示 |

---

### F022: 教室詳細ドリルダウン

#### 処理

一覧から教室を選択 → 右パネルに教室詳細をスライドイン表示

#### 出力

P002ダッシュボード（教室長）相当の情報を参照モードで表示。

| セクション | 表示内容 |
|-----------|---------|
| サマリー | 充足率、講師数、生徒数 |
| 充足率推移グラフ | 過去6ターム分の推移 |
| 人員ヒートマップ | 曜日×時限の講師/生徒配置状況 |
| 時間割サマリー | 現在の時間割概要 |
| 問題点リスト | 充足率低下要因等の注意事項 |

#### 制約

- **参照モードのみ**: 編集操作は不可
- 時間割の調整、マスタの編集等はできない
- 問題の特定と状況把握のみに使用

---

### F023: 問題教室アラート

#### アラート条件

| 条件 | アラートレベル | 表示 |
|------|--------------|------|
| 充足率60%未満 | 危険 | 赤バッジ |
| 前月比10ポイント以上低下 | 危険 | 赤バッジ |
| 充足率60〜79% | 警告 | 黄バッジ |
| 未確定の時間割が期限超過 | 警告 | 黄バッジ |

#### 出力

- 問題教室の強調表示（一覧上部に固定）
- アラート理由の表示

---

### F024: 教室比較

複数教室の指標を並べて比較表示する。

#### 入力

- 比較対象教室（2〜4教室をチェックボックスで選択）

#### 出力

| 比較項目 | 表示内容 |
|---------|---------|
| 充足率 | 各教室の充足率を横並びで表示 |
| 前月比 | 各教室の増減を横並びで表示 |
| 講師数/生徒数 | 人員状況の比較 |
| 充足率推移 | 複数教室の推移を重ねたグラフ |

#### 処理

1. 教室一覧でチェックボックスにより複数選択
2. 比較ボタンクリック
3. 比較モーダルを表示
4. 選択した教室の指標を横並びで表示

## 4. UI状態遷移

```
[マネジメントダッシュボード表示]
    ├─→ 教室行クリック → [教室詳細パネル（参照モード）]
    │                        ├─→ ×ボタン → [パネル閉じる]
    │                        └─→ パネル外クリック → [パネル閉じる]
    ├─→ チェックボックスで複数選択 → [比較ボタン有効化]
    │                                  └─→ 比較ボタンクリック → [比較モーダル表示]
    │                                                            └─→ 閉じる → [一覧へ]
    ├─→ アラートクリック → [該当教室詳細パネルへ]
    ├─→ フィルター/ソート → [一覧更新]
    └─→ 期間セレクター変更 → [一覧更新]
```

## 5. アクセス制御

| 役割 | アクセス可否 | 備考 |
|------|------------|------|
| 教室長 | × | - |
| エリアマネジャー | ○（担当エリアのみ） | 参照のみ、編集不可 |
| システム管理者 | ○（全エリア） | 参照のみ、編集不可 |

---

## 6. 「担当エリア」の定義

エリアマネジャーが参照できる教室の範囲は以下のように決定される。

| 設定場所 | 項目 | 説明 |
|---------|------|------|
| P012 ユーザー管理 | 担当エリアID | ユーザーに割り当てられたエリアID |
| P013 教室マスタ | 所属エリア | 各教室が所属するエリア |

**参照可能な教室**: ユーザーの担当エリアIDと、教室の所属エリアが一致する教室
