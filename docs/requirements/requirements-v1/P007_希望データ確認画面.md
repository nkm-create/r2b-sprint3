## 3. 機能詳細

### F060: 希望データマトリクス表示

曜日×限のマトリクスで希望値を表示。

#### 希望値の定義

| 値 | 講師での意味 | 生徒での意味 | 表示色 |
|----|------------|------------|--------|
| ○ | 勤務可能 | 受講希望 | 緑 |
| × | 勤務不可 | 受講不可 | 赤 |
| △ | - | できれば可 | 黄 |

#### 出力

- 縦軸: 講師名または生徒名
- 横軸: 曜日×時限のマトリクス
- セル: 希望値（○/×/△）

---

### F061: 講師/生徒希望切り替え

タブまたはトグルで表示対象を切り替え。

#### 切り替え対象

| タブ | 表示内容 |
|------|----------|
| 講師シフト希望 | 講師の出勤可能時間帯 |
| 生徒受講希望 | 生徒の通塾可能時間帯 |

---

### F062: 希望データ編集

セルクリックで値をトグル（○→×→△→○...）。編集済みセルは背景色で区別。

#### 操作

| 操作 | 動作 |
|------|------|
| セルクリック | 値をトグル（○→×→△→○...） |
| 編集済みセル | 背景色をハイライト |
| 保存ボタン | 変更を確定 |
| リセットボタン | 編集前の状態に戻す |

#### 出力

- 編集後の希望値がマトリクスに反映

---

### F063: Google Form同期

Google Sheets API経由でデータ取得 → 差分検出 → 手動編集済みデータの上書き確認 → 更新

#### 処理フロー

1. Google Sheets APIでスプレッドシートのデータを取得
2. 既存データとの差分を検出
3. 手動編集済みのセルがある場合、上書き確認ダイアログを表示
4. 確認後、データを更新

#### 出力

- 同期結果サマリー（追加/更新/削除件数）
- 同期日時の記録

---

### F064: 未回答者確認

希望データ未入力の講師・生徒をリスト表示。

#### 出力

| 項目 | 説明 |
|------|------|
| 種別 | 講師/生徒 |
| 氏名 | 未回答者の氏名 |
| 登録日 | マスタ登録日 |
| 希望データ状況 | 未入力/一部入力/入力済み |

## 4. UI状態遷移

```
[希望データ確認画面]
    ├─→ タブ切り替え → [講師/生徒マトリクス表示]
    ├─→ セルクリック → [値トグル] → [編集状態]
    │                                ├─→ 保存 → [確定]
    │                                └─→ リセット → [編集前に戻す]
    ├─→ Google Form同期ボタン → [同期処理]
    │                            ├─→ 差分なし → [完了メッセージ]
    │                            └─→ 差分あり → [上書き確認] → [更新]
    └─→ 未回答者確認ボタン → [未回答者リスト表示]
                               └─→ 行クリック → [該当者の希望データ編集画面へ]
```

## 6. 関連機能

### Google Form連携の設定

Google Form連携の初期設定（スプレッドシートURL、OAuth認証、カラムマッピング）は **P004 教室設定画面（F034）** で行う。

P007ではP004で設定済みの連携先からデータを同期する。
